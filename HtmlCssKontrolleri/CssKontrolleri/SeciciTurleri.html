Satır Bazında (inline css)

Şuana kadar css te sürekli olarak satır bazında css(inline css) kullandık.

satır bazında css etiket'e style ekledikten sonra css kodu eklenmesi ile yapılır.

Seçici Kullanımı

Secicileri tek tek satıra değil head etiketleri arasında style ile belirtilen css alanına yazıyoruz. Amacı her etikete ayrı ayrı css yazmayı engellemektir.

Etiket Secici

head blogundaki style etiketinin içerisine span{} yazılarak sayfadaki tüm span'lerin tamamının tek bir css kodu ile değişmesini sağlayabiliriz
Ben bir span'im
css'ler etiketler üzerine eklenerek birçok seçiciden veri alması sağlanabilir. Örneğin sayfadaki tüm yazılar öncelikle yeşil'di span'e eklenen css ler ile body deki tüm özellikler span'e geçmişken color,font-family ve font-size değiştirldi.

Sınıf Seçiciler

head bloğundaki style etiketi içerisinde tamımlanan sınıflar ile istenilen tüm etiketlere ayrı ayrı uygulanabilecek css yapısıdır.
Sınıflar "." ile başlayıp türkçe karakter kullanmadan istenilen isimde tanımlanır.
